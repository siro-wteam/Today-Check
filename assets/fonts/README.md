# Geist Font Files

이 디렉토리에 Geist 폰트 파일을 추가하세요.

## 다운로드 방법

1. GitHub에서 Geist 폰트 다운로드:
   - https://github.com/vercel/geist-font/releases
   - 또는 https://github.com/vercel/geist-font

2. 필요한 폰트 파일:
   - `Geist-Regular.ttf` (400)
   - `Geist-Medium.ttf` (500)
   - `Geist-SemiBold.ttf` (600)
   - `Geist-Bold.ttf` (700)

3. 다운로드한 폰트 파일을 이 디렉토리에 복사하세요.

4. 라이선스 파일:
   - `OFL.txt` 파일도 함께 복사하세요 (라이선스 준수)

## 폰트 파일 구조

```
assets/fonts/
  ├── Geist-Regular.ttf
  ├── Geist-Medium.ttf
  ├── Geist-SemiBold.ttf
  ├── Geist-Bold.ttf
  └── OFL.txt
```

## 라이선스

Geist 폰트는 SIL Open Font License 1.1 (OFL-1.1)로 라이선스되어 있습니다.
상업적 사용이 가능하며, 라이선스 파일을 포함해야 합니다.
